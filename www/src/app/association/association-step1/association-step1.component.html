Prérequis

Pour remplir cette étape, vous devez avoir activé le HTTPS sur votre freebox pour crypter le trafic qui transite entre votre box de notre système. Si vous n'etes pas familier avec ce procédé, je vous redirige vers les tutoriaux suivants :

  • https://www.freenews.fr/freenews-edition-nationale-299/freebox-9/lacces-distant-a-freebox-os-sameliore-https
  • http://www.universfreebox.com/article/33426/Tutoriel-Comment-acceder-a-Freebox-OS-depuis-n-importe-ou-de-maniere-securisee

Identification de la Freebox

  • Le nom de domaine est affiché sur cette interface : http://mafreebox.freebox.fr/#Fbx.os.app.settings.domains.Domains
  • Le port HTTPS est affiché sur cette interface : http://mafreebox.freebox.fr/#Fbx.os.app.settings.Accounts
